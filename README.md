# Data Portfolio

Personal portfolio project focused on data analytics, visualization and modern web technologies.

The goal of this project is to combine Python, SQL and interactive frontend components into a clean and visually understandable data application.

## Tech Stack

- Next.js
- React
- Tailwind CSS
- Framer Motion
- Recharts
- Vercel

## Features

- Responsive portfolio website
- Interactive dashboard components
- Animated UI elements
- Data-focused design
- Modern frontend architecture

## Local Development

Start the development server:

```bash
npm run dev -- --webpack

Then open:
